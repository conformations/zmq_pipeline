zmq_pipeline
============

A simple example implementing a multiple Producer, multiple Consumer paradigm in 0mq.

Prerequisites:
- gflags (http://code.google.com/p/google-gflags/)
- glog (http://code.google.com/p/google-glog/)
- zeromq (http://www.zeromq.org/)

Compiling:
- make

Running:
- ./broker
- ./sink
- ./worker
- ./source
